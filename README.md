# 🚗 BN Detailer --- Plataforma de Serviços de Lavagem

> **Soluções modernas para limpeza automotiva.**\
> A **BN Detailer** é uma aplicação frontend desenvolvida para
> apresentar serviços de lavagem e detalhamento automotivo, oferecendo
> uma experiência moderna, responsiva e profissional aos usuários.

Este repositório contém o **frontend da aplicação**, desenvolvido em
**React + Vite**, com foco em componentização, organização de código e
estilização modular com SCSS Modules.

------------------------------------------------------------------------

## ✨ Objetivo do Projeto

A plataforma BN Detailer foi criada com os seguintes objetivos:

-   Apresentar os serviços oferecidos pela empresa
-   Exibir galeria de veículos e trabalhos realizados
-   Facilitar o contato e engajamento com clientes
-   Oferecer uma experiência moderna e responsiva
-   Transmitir confiança e qualidade por meio do design

------------------------------------------------------------------------

## 🚀 Tecnologias Utilizadas

-   **React.js** --- Biblioteca para construção de interfaces
-   **Vite** --- Ferramenta de build e servidor de desenvolvimento
-   **SCSS (CSS Modules)** --- Estilização modular e organizada
-   **npm** --- Gerenciador de pacotes
-   **ESLint** --- Padronização e qualidade de código

> ❌ Não foram utilizados frameworks CSS ou bibliotecas de UI prontas.

------------------------------------------------------------------------

## 🧩 Funcionalidades

-   **Página Inicial** --- Apresentação dos serviços e chamada para ação
-   **Header Responsivo** --- Navegação e identidade visual
-   **Footer** --- Informações institucionais e links úteis
-   **Galeria de Imagens** --- Exibição de veículos e serviços
    realizados
-   **Componentização Modular** --- Estrutura reutilizável e escalável

## 📱 Responsividade

A aplicação foi desenvolvida utilizando boas práticas de design
responsivo:

-   Uso de unidades relativas (`%`, `rem`, `vh`, `vw`)
-   `max-width` para controle de largura
-   Media queries para diferentes breakpoints
-   Layouts flexíveis com Flexbox e CSS Grid
